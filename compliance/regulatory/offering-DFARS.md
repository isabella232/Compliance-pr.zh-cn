# <a name="defense-federal-acquisition-regulation-supplement-dfars"></a>防御联邦收购法规补充 (DFARS) 

## <a name="dfars-overview"></a>DFARS 概述

在2016年10月21日，美国国防部 (DoD) 颁发最终规则修正了国防联邦收购法规补充 (DFARS) ，并对其信息系统处理、存储或传输涉及的防御信息 (CDI) 的防御承包商施加保护和网络事件报告义务。  
  
最终的 DFARS 子句 252.204-7012 (受保护的防御信息和网络事件报告) 指定用于包括网络事件报告要求的安全措施以及云服务提供商的其他注意事项。 按照 DFARS 252.204-7012，所有 DoD 承包商和国防工业基座都必须遵守良好安全的 DFARS 要求，但不是在2017年12月31日之后。 '

## <a name="microsoft-and-dfars"></a>Microsoft 和 DFARS

Microsoft 政府云服务可帮助美国国防部工业和国防承包商客户满足应用于云服务提供商的 252.204-7012 的 DFARS 子句中枚举的 DFARS 要求。 当需要防御承包商遵守 DFARS 子句 252.204-7012 in 合同时，Microsoft 可以支持适用于 Azure 政府和 Office 365 美国政府防御服务的云服务提供商的要求。 这两种服务都演示了如何支持客户通过其 L5 资格鉴定和国防部安全要求指南中的 DFARS 7012 子句来满足所需的功能。

## <a name="frequently-asked-questions"></a>常见问题解答

**Microsoft Azure 政府和 Office 365 美国政府防御部门支持哪些 DFARS 要求？**

Azure 政府和 Office 365 美国政府防护允许我们的防御行业基础和国防承包商客户满足在 252.204-7012 的 DFARS 子句中枚举的 DFARS 要求，这些条款适用于云服务提供商。

**是否有独立的评估员验证了 Azure 政府和 Office 365 美国政府防御部门是否支持 DFARS 要求？**

是的，第三方评估组织已证明 Azure 政府和 Office 365 美国政府防护云服务产品符合 DFARS 子句 252.204- (7012 的适用要求。) 保护未分类控制的技术信息。

**受控制的未分类信息 (CUI) 和覆盖的防御信息 (CDI) 之间的关系是什么？**

CUI 是需要根据法律、法规或政府范围内的策略来保护或传播控件的信息。 [CUI 注册表](https://www.archives.gov/cui/registry/category-list.html)标识已批准的 CUI 类别和子类别。

CDI 是控制的技术信息或其他信息 (，如 CUI 注册表) 中所述，它需要保护或传播控件，也可以是：

- 在合同、任务顺序或交付订单中进行标记或以其他方式标识，并通过代表 DoD 或代表与合同性能的规定提供给合同工或
- 由承包商收集、开发、接收、传输、使用或存储这些条款，以支持合同的性能

**所有 Microsoft 服务是否都符合 DFARS 法规下的 "已覆盖的防御信息" 的 "足够的安全性" 要求？**

2016年10月，美国国防部 (DoD) promulgated a 最终规则实施防御联邦获取法规补充 (DFARS) 条款，这些条款适用于处理、存储或传输 "涉及的防御信息" 的所有 DoD 承包商。 该规则指出，此类系统必须满足 NIST SP 800-171 中规定的安全要求， [保护 nonfederal 信息系统和组织中的受控制的未分类信息](https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-171.pdf)，或由 DoD 合同专员批准的 "备用的、但同样有效的安全措施"。 此外，DoD 承包商使用外部云服务提供商来处理、存储或传输涵盖的防御信息，此类提供程序必须满足与 FedRAMP 中等比较基准等效的安全要求。

以下 Microsoft 云服务已收到 FedRAMP 中型授权，适用于 DFARS： Azure 政府、Dynamics 365 美国政府、Office 365 美国政府和 Office 365 美国政府防御部门。

此外，在 FedRAMP 认证边界之外的其他产品（由 DoD 承包商在处理、存储或传输 "覆盖的防御信息"）中，我们正在进行审阅，以满足2017年12月31日的合规性截止日期的要求。 Microsoft 致力于记录这些内部和面向客户的服务如何遵守 NIST SP 800-171 或可接受的安全等效项，以满足 DFARS 相关条款的要求。
